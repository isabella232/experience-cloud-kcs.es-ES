# Error al acceder al perfil

## Descripción


Error al acceder al perfil 

BAS-010040 No se puede convertir &#39;cusFieldbyTest&#39; (texto de tipo) a tipo entero (32 bits) (bAS-010011 Valor no válido)


## Resolución


Reparar la base de datos mediante la opción reparar base de datos y debería resolver el problema
